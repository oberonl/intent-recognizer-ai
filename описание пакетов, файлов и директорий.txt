Проект имеет классическую для Python приложений структуру пакетов:
    В корне проекта лежит дириектория src содержащая в себе исходный код проекта
    В директории docs находится документация для проекта
    Файл about.txt - содержит общее высокоуровневое описание алгоритма работы проекта
    Файл readme.md - содержит описание процесса сборки и запуска приложения
    Файл run.sh - содержит заранее подготовленный файл-команду, через которую можно запустить приложение
    Файл therms.txt - содержит в себе термины, используемые в описании (коментарии) данного проекта
    Файл used-tech.txt - содержит список используемых технологических решений проекта (библиотеки и фреймворки)
    Файл описание пакетов, файлов и директорий.txt - текущий файл

Директория src имеет типовую структуру для Python проектов

├───src - содержит исходные файлы проекта
    ├───model - содержит код использования модели машинного обучения
    ├───main.py - основной метод, содержит логику запуска + содержит API
